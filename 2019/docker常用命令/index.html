docker常用命令

Written by Alfred Young on July 22, 2019 in docker |

      • 查看容器对象的底层基础信息
      • 批量删除无标签的镜像
      • 进入容器内部
      • 查看容器输出日志
      • 查看容器资源使用情况

查看容器对象的底层基础信息

docker inspect <容器id or 容器name >

批量删除无标签的镜像

开发及测试环境时部署代码时每次都会自动生成同一标签的镜像，旧的镜像标签会被移除而显示为<none>
docker rmi -fdocker images | grep ‘’ | awk ‘{print $3}’``

进入容器内部

docker exec -it <容器id or 容器name> /bin/sh

查看容器输出日志

docker logs -f --tail 300 <容器id or 容器name>

查看容器资源使用情况

docker stats <容器id, 不输入则显示全部>
